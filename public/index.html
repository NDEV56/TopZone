Langkah 1 / 5 — Pilih Cara Membagikan Server ke Internet

"Tunnel" adalah teknologi yang membuat server lokal di komputermu bisa dibuka dari HP atau komputer lain di luar rumah. Pilih satu di bawah. Kalau bingung, pilih saja yang nomor 1 (ngrok).

Langkah 2 / 5 — Masukkan Token Ngrok

📋 Cara dapat token (5 menit, gratis):
  1. Buka link ini di browser baru: dashboard.ngrok.com/signup
  2. Daftar pakai email atau Google/GitHub.
  3. Setelah login, buka: Your Authtoken
  4. Klik tombol "Copy" untuk salin token.
  5. Tempel di kotak di bawah ini ↓
Token tidak akan terlihat. Disimpan di file .env di komputermu. Kami tidak pernah mengirimnya ke mana-mana.
🛠️ Pengaturan lanjutan (boleh dilewati)
Hanya kalau kamu sudah punya domain ngrok dari paid plan atau free static domain. Kosongkan kalau tidak tahu.

Langkah 3 / 5 — Pilih Aplikasi Server Lokal

Server lokal adalah aplikasi yang menjalankan PHP di komputermu. Kalau kamu sudah pakai XAMPP/Laragon/WAMP, pilih sesuai itu. Kalau belum, pilih Auto-detect.

🔎 Memindai komputermu untuk server yang sudah terinstall…
🛠️ Pengaturan PHP Built-in (kalau pilih 'php')
Folder yang akan dijadikan webroot. Default: folder Home/ di project ini.

Langkah 4 / 5 — Keamanan Panel Kontrol

Panel kontrol ini hanya bisa diakses dari komputer ini saja secara default. Jadi password tidak wajib. Tapi kalau kamu ingin akses dari HP di Wi-Fi yang sama, isi password yang kuat.

Minimal 6 karakter. Disimpan dengan hash (scrypt) — tidak bisa dibalik jadi password asli, bahkan kalau file .env bocor.
Kalau temanmu push update di GitHub, sistem akan beri tahu. Pilih "Pull otomatis" hanya kalau kamu yakin update selalu aman.

Langkah 5 / 5 — Konfirmasi

Cek ringkasan di bawah. Kalau ada yang salah, kembali ke langkah sebelumnya. Kalau sudah benar, klik tombol Simpan & Mulai.

💡 Setelah simpan, panel ini akan langsung berusaha menyalakan server-mu. Kalau gagal — tenang, pesan error akan muncul dengan langkah perbaikan.

📅 Auto-cleanup Terjadwal (Opsional)

Kamu bisa menambahkan task ke crontab/Task Scheduler untuk membersihkan otomatis tiap minggu:

Linux/Mac (crontab): 
0 3 * * 0 cd "/path/ke/TopZone" && node server.js --clean --apply >> logs/cleanup-cron.log 2>&1
Windows (PowerShell): 
schtasks /create /tn "TopZoneClean" /tr "node server.js --clean --apply" /sc weekly /d SUN /st 03:00

🚨 Emergency Lockdown

Aktifkan kalau panel kontrol sedang dalam serangan berat. Mode guarded cuma memperketat limit; restricted mematikan semua aksi state-changing kecuali login/logout; lockdown mematikan semuanya kecuali endpoint health.

⚠️ Cara kerja auto-trigger: kalau banyak indikator serangan (SQLi attempt, brute force, RPS spike, banyak IP berbeda) tertangkap dalam 60 detik, sistem otomatis naik tingkat — mulai dari guarded → restricted → full lockdown. Mode normal kembali setelah 2 menit "tenang".

❓ Panduan untuk Pemula

Pertanyaan paling sering ditanyakan:

🎯 Bagaimana cara mulai server?
  1. Buka XAMPP / Laragon / WAMP. Klik tombol Start di baris Apache. Tunggu sampai berwarna hijau.
  2. Kalau pakai database, klik Start juga di MySQL.
  3. Kembali ke panel ini, buka tab Kontrol.
  4. Klik tombol ▶ Mulai Server.
  5. Tunggu 5-15 detik. URL publik akan muncul di Dashboard.
  6. Salin URL itu untuk dibagikan ke teman / Midtrans.
❌ Server gagal start, apa yang harus dilakukan?
  1. Buka tab Diagnostik → klik Pindai Ulang.
  2. Lihat baris berwarna merah → ikuti petunjuk di sebelahnya.
  3. Cek tab Log → filter "Error" atau "Kritis".
  4. Pastikan XAMPP/Laragon benar-benar START (bukan cuma terbuka).
  5. Coba ganti port — kadang port 80 dipakai aplikasi lain (Skype, IIS).
🔑 Token Ngrok di mana dapat?
  1. Buka dashboard.ngrok.com/signup
  2. Daftar gratis (cukup email).
  3. Setelah login, buka menu Your Authtoken.
  4. Klik tombol Copy.
  5. Tempel di tab Pengaturan → field "Token Ngrok".
  6. Klik Simpan, lalu Restart.
🔄 Bagaimana update kalau temanku push ke GitHub?
  1. Buka tab Update.
  2. Klik tombol Cek Update.
  3. Kalau ada commit baru, ringkasannya muncul.
  4. Klik Tarik Update.
  5. Backup otomatis dibuat → kalau gagal, otomatis rollback.
  6. Restart server setelah update selesai.
🐌 Tunnel ngrok lambat, ada alternatif?

Buka Pengaturan → ganti Provider ke cloudflared atau localtunnel. Aktifkan "Otomatis pakai tunnel lain kalau gagal" supaya kalau yang dipilih down, sistem coba yang lain.

📱 Bisa diakses dari HP?

Tentu. URL publik (yang ada .ngrok-free.app atau .trycloudflare.com) bisa dibuka dari mana saja — HP, laptop teman, dst.

🛡️ Apakah aman?
  • Token kamu disimpan di file .env di komputermu sendiri.
  • Panel ini default hanya bisa diakses dari 127.0.0.1.
  • Setiap request mencurigakan (SQL injection, path traversal, dll) langsung diblokir & dicatat di log Keamanan.
  • Brute force login → IP otomatis diblokir setelah 5 percobaan.
💼 File apa yang penting?
  • .env — konfigurasi (token, mode, port).
  • server.js — launcher CLI.
  • gui.js — panel ini.
  • lib/ — modul-modul internal.
  • logs/ — log harian per kategori.
  • backups/ — snapshot sebelum tiap auto-update.
  • Home/ — folder website PHP-nya.
